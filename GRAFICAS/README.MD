GRAFICA_EN_MOVIMIENTO; ANIMATED_GRAPH

History = Historia:
Sometimes, it is required to graph a function that progresses along the x-axis.
A veces, se requiere graficar una funcion que se despliegue en funcion del eje x.

What the project does = Que hace el proyecto:
Movimiento_sen allows you to generate a gif with the animated 2D equation as a function of the x-axis.
Movimiento_sen Permite generar un gif con la ecuacion animada en 2D en funcion del eje x.

Why the project is useful = Para que podría servir el proyecto:
It allows you to embed an animated gif into a presentation or video.
Permite incrustar un gif animado en una presentacion o video.
